## Latest emitted Events on Velas network

This query retrieves the latest emitted events on Velas network.
You can run the query [here](https://ide.bitquery.io/latest-emitted-events-on-velas#)

```
{
  ethereum(network: velas) {
    smartContractEvents(
      options: {desc: "block.timestamp.iso8601", limit: 10}
      date: {after: "2024-08-01T00:00:00Z"}
    ) {
      block {
        timestamp {
          iso8601
        }
      }
      smartContractEvent {
        name
        signature
        signatureHash
      }
      transaction {
        hash
      }
    }
  }
}

```
